# The Tarragrue

Both tanks should try to get [Ever-Beating Heart](https://www.wowhead.com/spell=347966/ever-beating-heart)

## Phase 1 (100% - 10%)

Swap on each [Overpower](https://www.wowhead.com/spell=346985/overpower)

### Heroic Only

Don't soak [Remnant: Upper Reaches' Might](https://www.wowhead.com/spell=352382/remnant-upper-reaches-might)

Can soak [Remnant: Mort'regar's Echoes](https://www.wowhead.com/spell=352389/remnant-mortregars-echoes) and [Remnant: Soulforge Heat](https://www.wowhead.com/spell=352398/remnant-soulforge-heat)

## Intermission

[Hungering Mist](https://www.wowhead.com/spell=347671/hungering-mist) run to clear area

## Phase 2 (10% - 0%)

Boss stops abilites and does 500% increased damage

Taunt boss between tanks and kite
